# Cinema Ticket Management System (Modular Programming)

## Overview
This project is a console-based cinema ticket management system developed using modular programming principles.

## Features
- Movie selection
- Seat booking system
- Ticket price calculation
- Payment processing
- Admin management system

## Programming Concepts Used
- Modular programming
- Function decomposition
- File handling (if used)
- Input validation

## Language Used
C++ 

## System Design
Each module is separated into different functions to ensure maintainability and scalability.
